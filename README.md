# 💊 Medicare Pharmacy App  

## 📱 Features  
- Add New Stock  
- Check Stock  
- Automated Testing

- Language: Java  
- Framework: Android SDK  
- Testing: Espresso, JUnit

## 📂 Project Structure  
- app/  
- gradle/  
- tests/  

## ▶️ How to Run  
- Clone Repository  
- Open in Android Studio  
- Sync Gradle  
- Run on Emulator/Device
